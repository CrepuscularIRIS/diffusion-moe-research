# Fable5 Methodology Audit

This folder contains the 11 methodology review files produced by Fable5, plus a local quality review.

## Files

| File | Purpose |
| --- | --- |
| `fable5-methodology-00-reading-map.md` | Maps source documents, overlaps, missing document types, and stale entry points. |
| `fable5-methodology-01-evidence-standards.md` | Defines evidence grades, claim types, HOLD rules, and escalation boundaries. |
| `fable5-methodology-02-research-action-selection.md` | Chooses between search, falsifier, review, revision, compression, promotion, and stopping. |
| `fable5-methodology-03-kill-rescope-tradeoffs.md` | Defines KILL / RESCOPE / HOLD / DO boundaries and conversion rules. |
| `fable5-methodology-04-commitment-risk.md` | Audits premature commitments and requires delay cost plus reversible substitutes. |
| `fable5-methodology-05-agent-governance-tradeoffs.md` | Defines role boundaries, contamination risks, and governance overhead tradeoffs. |
| `fable5-methodology-06-artifact-acceptance.md` | Reviews artifact readiness across papers, reports, README, demos, benchmarks, skills, and logs. |
| `fable5-methodology-07-cross-domain-transfer.md` | Transfers research operators into product, engineering, design, education, business, and organization contexts. |
| `fable5-methodology-08-execution-boundaries.md` | Separates automatable gates from judgment-heavy cores. |
| `fable5-methodology-09-calibration-cases.md` | Provides 20 calibration cases and a scoring sheet. |
| `fable5-methodology-10-final-methodology.md` | Provides the compact final methodology, decision tree, role table, verdict table, and anti-patterns. |
| `quality-review.md` | Local assessment of whether the output meets the bar. |

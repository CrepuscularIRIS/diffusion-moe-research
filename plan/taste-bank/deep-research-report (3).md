# 建模品味图谱深度研究

## 研究口径与筛选说明

这份报告**不把“换 backbone、加模块、做 benchmark、提一点 SOTA”算作对象转移**。我只保留那些真正把“研究者默认在建模什么”改写掉的工作：把状态换成算子、把界面换成场、把联合分布换成因子图、把单次求解换成多分辨残差、把非线性轨迹换成线性可观测演化，等等。为了不把味道局限在近年的 SciML，我刻意把**经典数值分析、统计力学、控制、编码理论、图模型、最优传输**和近年的 operator learning 放在同一张图上，因为后者很多最有迁移价值的动作，其实来自前者。关于 operator learning、fluid mechanics、operator inference 等大类的历史与影响，可参考近年的综述与年度回顾。citeturn0search3turn0search11turn0search19turn3search3

下面共整理 **51 个案例**。其中“旧对象→新对象”以及“压缩了什么复杂性”是我基于原论文做的**分析性抽象**；这部分并非总是论文原句，但都尽量贴着方法的真正操作点写，而不是贴着宣传语写。

## 几何与离散化对象转移

| 案例 | 关键信息 | 可靠来源 |
|---|---|---|
| **Symplectic integrators / 几何数值积分** | **领域**：哈密顿动力系统、分子动力学、天体力学。**代表**：Hairer, Lubich, Wanner，《Geometric Numerical Integration》，2002/2006。**旧对象**：逐时刻的数值状态点与局部截断误差。**新对象**：近似解析流的**辛映射**与其对应的**modified Hamiltonian**。**数学载体**：symplectic map、backward error analysis。**压缩/重组复杂性**：把“长期能量漂移、体积失真、轨道拓扑走样”从很多局部误差细节里抽出来，重组为“是否保持几何结构”。**新能力**：能长期预测和控制能量壳、相空间体积、准周期结构，而不是只优化短时状态误差。**为何有影响**：它把“求轨迹”改成“求保结构流”，成为后续 Hamiltonian ML、可逆网络、长期 rollout 稳定性的原型 taste。**迁移价值**：高，可迁到 VLA world model、diffusion/LLM 的守恒更新、systems 中长期稳定 rollout。 | 论文/书与综述：citeturn11search0turn11search9 |
| **Variational integrators / 离散变分积分** | **领域**：几何力学、控制、机器人。**代表**：Marsden & West, 2001, *Acta Numerica*。**旧对象**：离散更新公式本身。**新对象**：离散作用量上的**变分原理**。**数学载体**：discrete Lagrangian、discrete Euler–Lagrange equation。**压缩/重组复杂性**：将约束、动量守恒、时间可逆性等复杂细节统一吸收到离散作用量的设计中。**新能力**：天然保动量、易处理约束与最优控制耦合，能直接把控制问题写成“在离散作用量上优化”。**为何有影响**：它把积分器从“代数格式”升级成“物理原理离散化”，影响机器人、航天、软体系统建模。**迁移价值**：高，可迁到 VLA 规划器、具物理先验的 policy rollout、多模态控制器的约束一致性。 | 论文/讲义：citeturn11search1turn11search7turn11search4 |
| **Godunov / Finite Volume Methods** | **领域**：双曲守恒律、CFD。**代表**：Godunov 家族；现代基础综述见 Barth–Herbin–Ohlberger。**旧对象**：节点上的点值解。**新对象**：控制体上的**cell average** 与穿越边界的**numerical flux**。**数学载体**：conservation law in integral form、Riemann solver、flux balance。**压缩/重组复杂性**：把 shock、间断、守恒约束从难以稳定的点值表示，转移到边界通量这个更局部、可拼接的对象上。**新能力**：能稳定预测冲击波位置、守恒量、波系相互作用；旧的点值中心差分很难直接控制这些量。**为何有影响**：现代 CFD、大量高分辨率格式都建立在“平均量+通量”对象转移上。**迁移价值**：高，可迁到 diffusion/LLM 的通量式 token transport、systems 中局部守恒式调度、dataset evaluation 的分布迁移度量。 | 教材/综述：citeturn10search15turn10search3 |
| **Level Set Method** | **领域**：界面运动、流体、几何演化、图形学。**代表**：Osher–Sethian 传统；Sethian 1999/2001。**旧对象**：显式跟踪的界面点、前沿网格。**新对象**：全空间上的**signed distance / level-set function**。**数学载体**：隐式函数、Hamilton–Jacobi PDE。**压缩/重组复杂性**：把拓扑变化、并裂、重连、界面法向与曲率等复杂操作吸收到一个标量场的零等值面里。**新能力**：可直接测量曲率、法向速度和拓扑变化；旧式 front tracking 很难在并裂时维持鲁棒性。**为何有影响**：它把“几何对象”改写成“函数对象”，此后大量 free-boundary 问题都可借 PDE machinery 来做。**迁移价值**：高，可迁到 VLA 的可交互边界表示、MLLM 的对象轮廓潜变量、3D generation 的拓扑稳定表示。 | 书/综述：citeturn8search8turn8search11turn8search4 |
| **Phase-Field / Cahn–Hilliard diffuse interface** | **领域**：相变、材料、两相流。**代表**：Cahn–Hilliard 1958 传统；现代综述见 2021/2022 review。**旧对象**：sharp interface 与显式移动边界。**新对象**：连续的**order parameter / phase field**。**数学载体**：自由能泛函、梯度流、Cahn–Hilliard / Allen–Cahn PDE。**压缩/重组复杂性**：把界面几何与拓扑演化压进自由能与弥散界面宽度里，避免显式追踪边界。**新能力**：可直接优化界面能、成核、粗化、拓扑变换；旧范式难以统一处理复杂界面网络。**为何有影响**：相场法成为材料微结构计算和多相流的通用中介对象。**迁移价值**：高，可迁到多模态融合中的软边界表示、VLA manipulation 的接触/相变近似、dataset evaluation 的连续结构描述。 | 综述/材料：citeturn8search1turn8search12turn8search16 |
| **Lattice Boltzmann Method** | **领域**：流体力学、复杂边界、微流控。**代表**：McNamara & Zanetti, 1988, *PRL*。**旧对象**：直接离散 Navier–Stokes 的宏观速度/压力场。**新对象**：格点上的**粒子分布函数**。**数学载体**：discrete Boltzmann equation、BGK collision、moments。**压缩/重组复杂性**：把连续压力-速度耦合、椭圆约束等难点重组为局部碰撞+平移的消息传播。**新能力**：可在复杂边界、多相、多孔介质里高效模拟且自然并行；旧对象下的求解往往更难局部化。**为何有影响**：它把流体从“场方程”改成“分布演化”，在 HPC、复杂几何流体和 differentiable simulation 中都很耐用。**迁移价值**：高，可迁到 token transport、局部消息传播式 world model、systems 中可并行的局部更新器。 | 原始论文与回顾：citeturn11search2turn11search12turn8search6 |
| **Immersed Boundary Method** | **领域**：流固耦合、生物流体。**代表**：Peskin, 2002, *Acta Numerica*。**旧对象**：为复杂边界重新生成 body-fitted mesh。**新对象**：固定 Eulerian 流场网格 + 移动的 **Lagrangian boundary markers**。**数学载体**：Eulerian–Lagrangian coupling、smoothed delta kernel。**压缩/重组复杂性**：把复杂几何网格生成与重网格化的代价，转移成规则网格上的耦合核。**新能力**：能直接模拟心脏瓣膜、纤维、膜结构与流场相互作用。**为何有影响**：它把几何复杂性从网格几何转移到耦合算子，是现代生物/软体流固耦合的核心 taste。**迁移价值**：高，可迁到 VLA 接触建模、视觉-动作中的对象/场耦合、多模态物理交互建模。 | 论文与讲义：citeturn9search0turn9search4turn9search16 |
| **Discrete Exterior Calculus** | **领域**：几何计算、场论离散化、电磁/流体。**代表**：Desbrun, Hirani, Leok, Marsden, 2005。**旧对象**：坐标化的向量场分量与 ad hoc 差分模板。**新对象**：网格上的**cochains / discrete differential forms**。**数学载体**：simplicial complex、Hodge star、exterior derivative。**压缩/重组复杂性**：把坐标依赖、网格方向性与算子兼容性问题，压缩成离散拓扑与代数结构。**新能力**：可稳定表达守恒、Stokes 定理、拓扑约束和 gauge 结构。**为何有影响**：DEC 是“把连续几何换成离散代数拓扑对象”的经典模板。**迁移价值**：高，可迁到图神经 operator、3D scene 表示、MLLM 中结构一致性约束。 | 原始论文与历史综述：citeturn2search2turn2search17turn2search20 |

## 粒子 非局部 层级与自由能对象转移

| 案例 | 关键信息 | 可靠来源 |

| **Smoothed Particle Hydrodynamics** | **领域**：无网格流体、天体物理、自由表面。**代表**：Gingold–Monaghan / Lucy 1977 传统；Monaghan 1992 review。**旧对象**：固定网格上的场变量。**新对象**：携带质量/动量的**拉格朗日粒子云**。**数学载体**：kernel density estimate、particle measure。**压缩/重组复杂性**：把移动边界、大变形、自由表面等几何复杂性转移到粒子邻域核估计。**新能力**：自然处理飞溅、破碎、自由表面与大位移。**为何有影响**：它把连续介质转成“测度上的局部统计”，后来反复回到 differentiable particles、graphics、robotics mixing problems。**迁移价值**：高，可迁到 VLA 可变拓扑场景、点云世界模型、systems 中粒子化资源流。 | 综述与历史来源：citeturn10search0turn10search2turn10search5 |
| **Material Point Method** | **领域**：大变形固体、地质、雪/布料模拟。**代表**：Sulsky, Zhou, Schreyer, 1995。**旧对象**：纯 Lagrangian 网格或纯 Eulerian 网格。**新对象**：携带历史变量的**material points** + 临时背景网格。**数学载体**：particle-grid hybrid state。**压缩/重组复杂性**：避免 Lagrangian mesh 扭曲，也避免 Eulerian 界面扩散，把历史依赖压在粒子上、求解稳定性放在背景网格上。**新能力**：可模拟碰撞、大塑性、断裂前后过渡、多材料接触。**为何有影响**：它是“状态存在粒子上，约束求解发生在网格上”的混合对象范式。**迁移价值**：高，可迁到可微仿真、VLA 接触任务、机器人中的 hybrid state estimation。 | 初始与应用材料：citeturn9search6turn9search21turn9search18 |
| **Peridynamics** | **领域**：断裂力学、非局部固体。**代表**：Silling, 2000。**旧对象**：局部应力-应变 PDE。**新对象**：材料点之间的**nonlocal bond interaction**。**数学载体**：integral equation over neighborhoods、bond-based state。**压缩/重组复杂性**：把裂纹尖端奇异性和导数失效问题，从局部微分结构转移为非局部积分相互作用。**新能力**：能直接表示裂纹萌生、扩展、分叉，不必预先指定裂纹面。**为何有影响**：它给出一个“用非局部算子替换局部导数”的极强 taste。**迁移价值**：高，可迁到 OOD 结构破坏、视觉接触中的断裂预测、多模态长期安全建模。 | 原始论文与综述性资料：citeturn9search1turn9search17turn9search9 |
| **Multigrid** | **领域**：椭圆 PDE、数值线性代数。**代表**：Brandt 传统；现代教程 Briggs–Henson–McCormick。**旧对象**：单一分辨率上的线性系统误差。**新对象**：跨尺度的**residual/error hierarchy**。**数学载体**：restriction、prolongation、coarse-grid correction。**压缩/重组复杂性**：把“高频误差好消、低频误差难消”的复杂谱结构拆开到不同网格层级。**新能力**：能近似实现与网格规模无关的收敛率，这在旧的一层迭代法中很难。**为何有影响**：它把求解对象改成“层级误差流”，极大影响 AMG、神经 multiscale operator、hierarchical planners。**迁移价值**：极高，可迁到 speculative decoding 的多尺度校正、LLM system scheduling、multimodal coarse-to-fine fusion。 | 教材与概述：citeturn16search4turn16search12turn16search14 |
| **Algebraic Multigrid** | **领域**：稀疏线性系统、非结构网格。**代表**：Ruge & Stüben, 1987。**旧对象**：依赖几何网格的层级。**新对象**：由矩阵本身诱导的**algebraic coarse variables / interpolation graph**。**数学载体**：system matrix、strength-of-connection graph、interpolation operator。**压缩/重组复杂性**：去掉对显式几何的依赖，把多尺度结构直接从线性代数对象中提出来。**新能力**：把 multigrid 推广到黑盒稀疏矩阵与复杂工业离散。**为何有影响**：AMG 是“从算子本身反推建模层级”的经典长尾 operator。**迁移价值**：极高，可迁到 graph pooling、dataset condensation、systems optimization 中从依赖图自动抽 coarse variables。 | 原始来源与介绍：citeturn16search3turn16search11turn16search15 |
| **Wang–Landau sampling** | **领域**：统计力学、自由能计算。**代表**：Wang & Landau, 2001, *PRL*。**旧对象**：固定温度下的 Boltzmann 采样轨迹。**新对象**：系统的**density of states**。**数学载体**：non-Markovian random walk、log DOS estimate。**压缩/重组复杂性**：把温度依赖的采样困难转移成对 DOS 的统一估计。**新能力**：可直接得到熵、自由能和多温度热力学量，跨越粗糙能景。**为何有影响**：它把“采样轨迹”换成“态密度对象”，是很多增强采样算法的祖先味道。**迁移价值**：中高，可迁到 dataset evaluation 的 coverage object、diffusion LLM 的能量景探索、search over discrete programs。 | 原始论文：citeturn18search0turn18search4turn18search12 |
| **Metadynamics** | **领域**：分子模拟、稀有事件、自由能面。**代表**：Laio & Parrinello, 2002。**旧对象**：原始坐标上的长时间轨迹。**新对象**：少量**collective variables** 上的**bias potential / free-energy surface**。**数学载体**：CVs、history-dependent potential。**压缩/重组复杂性**：把高维稀有跃迁压缩到低维集体变量空间，并用累积偏置把势阱“填平”。**新能力**：能直接测量过渡态、自由能垒和反应通道。**为何有影响**：它提供了“先选低维 slow object，再在其上做主动探索”的通用配方。**迁移价值**：极高，可迁到 VLA 的技能探索、RL credit assignment、LLM test-time search over latent plans。 | 原始与现代综述：citeturn17search5turn18search18turn18search24 |
| **Persistent Homology** | **领域**：拓扑数据分析、材料结构、复杂几何。**代表**：Edelsbrunner, Letscher, Zomorodian, 2002。**旧对象**：单尺度几何特征或手工描述子。**新对象**：跨尺度的**birth–death topological features**。**数学载体**：filtration、persistence diagram / barcode。**压缩/重组复杂性**：把高维点云/结构中的多尺度孔洞、通道、环路压成稳定的拓扑签名。**新能力**：可测量结构稳健性、尺度依赖拓扑与微结构指纹；旧对象难以稳定比较。**为何有影响**：它把“形状”改写成“持久拓扑事件”，近年在材料、化学与几何学习里持续回流。**迁移价值**：高，可迁到 dataset evaluation、OOD 检测、multimodal latent topology、systems anomaly detection。 | 原始与应用综述：citeturn17search2turn17search10turn18search7 |

## 动力系统 控制与降阶对象转移

| 案例 | 关键信息 | 可靠来源 |

| **Adjoint method in PDE-constrained optimization** | **领域**：反问题、最优控制、逆设计。**代表**：方法族；现代综述 Allaire 2015。**旧对象**：逐参数有限差分灵敏度。**新对象**：约束系统对应的**adjoint state**。**数学载体**：Lagrangian dual、adjoint PDE。**压缩/重组复杂性**：把高维参数梯度计算从“每个参数一次前向求解”压缩到“一次前向 + 一次伴随”。**新能力**：可优化上百万参数的形状、材料或源项；旧对象几乎不可扩展。**为何有影响**：这是 inverse design、topology optimization、现代 differentiable simulation 的基底。**迁移价值**：极高，可迁到 VLA 反向规划、systems differentiable scheduling、LLM test-time control of latent objectives。 | 综述与教程：citeturn19search3turn19search19turn19search7 |

| **VAMPnets** | **领域**：分子动力学、深度动力学表示。**代表**：Mardt et al., 2018, *Nature Communications*。**旧对象**：先手工特征、再建 MSM。**新对象**：端到端学习的**metastable representation**。**数学载体**：deep neural networks optimizing VAMP objective。**压缩/重组复杂性**：把特征工程与状态划分合并成一个变分目标。**新能力**：能直接从原始输入学习慢态与动力学分解。**为何有影响**：它不是简单“深度化”，而是把建模对象稳定在“慢动力学表示”上。**迁移价值**：高，可迁到 video event segmentation、VLA skill discovery、LLM latent regime identification。 | 原始论文：citeturn17search0turn17search16 |
| **Information Bottleneck** | **领域**：信息论、表征学习。**代表**：Tishby, Pereira, Bialek, 1999/2000。**旧对象**：直接保留全部输入信息。**新对象**：关于目标最相关的**compressed representation**。**数学载体**：mutual information tradeoff、bottleneck variable。**压缩/重组复杂性**：把表示学习重写成“压缩无关信息、保留相关信息”的统计决策对象。**新能力**：可定量讨论表示中哪些信息该保留。**为何有影响**：很多 modern representation learning 都在重复这个对象转移。**迁移价值**：极高，可迁到 MLLM fusion、dataset summarization、distillation、systems telemetry compression。 | 原始论文与后续说明：citeturn12search2turn12search6turn12search17 |
| **Factor Graphs + Sum-Product** | **领域**：概率推断、编码、图模型。**代表**：Kschischang, Frey, Loeliger, 2001。**旧对象**：整体联合分布。**新对象**：局部分解的**factor graph + messages**。**数学载体**：bipartite graph、sum-product messages。**压缩/重组复杂性**：把指数级联合推断压成局部消息传递。**新能力**：可在树上精确、在 loopy graph 上近似地做边缘化与 MAP 推断。**为何有影响**：它是“把全局函数换成局部因子对象”的超级母体。**迁移价值**：极高，可迁到 agent systems、multimodal reasoning graphs、LLM tool-augmented planning。 | 原始论文：citeturn13search0turn13search4turn23search6 |
| **Tanner Graph Codes** | **领域**：编码理论、图码。**代表**：Tanner, 1981。**旧对象**：整体码本或单块校验矩阵。**新对象**：码约束的**bipartite graph representation**。**数学载体**：Tanner graph。**压缩/重组复杂性**：把全局编码约束重组为局部连接模式。**新能力**：使迭代译码与图推断成为可能。**为何有影响**：图码革命的关键是先换对象，再有后来的 LDPC/消息传递。**迁移价值**：高，可迁到 graph-structured memory、distributed verification、systems dependency coding。 | 原始论文：citeturn23search0turn23search8 |
| **LDPC Codes** | **领域**：编码理论。**代表**：Gallager, 1962。**旧对象**：密集码/一般线性码。**新对象**：**sparse parity-check graph**。**数学载体**：low-density parity-check matrix、sparse factor graph。**压缩/重组复杂性**：把码结构稀疏化，使局部迭代译码成为现实。**新能力**：在接近 Shannon 极限的同时保持可扩展解码。**为何有影响**：LDPC 说明“把对象换成稀疏图”本身就能改写算法可行性。**迁移价值**：高，可迁到 sparse attention、speculative verification、distributed system recovery。 | 原始论文与资料：citeturn23search1turn23search5turn23search17 |
| **Turbo Codes** | **领域**：编码理论、迭代推断。**代表**：Berrou, Glavieux, Thitimajshima, 1993。**旧对象**：一次性全局译码。**新对象**：两个编码器之间交换的**extrinsic information**。**数学载体**：iterative message passing between constituent decoders。**压缩/重组复杂性**：把难的全局译码重组为两个子系统之间的软信息循环。**新能力**：在合理复杂度下逼近 Shannon 极限。**为何有影响**：它把“译码对象”改成可往返更新的软信息，几乎预演了后来的迭代 inference。**迁移价值**：高，可迁到 self-refinement LLM、multi-agent speculative decoding、iterative multimodal reconciliation。 | 原始论文：citeturn13search1turn23search3turn23search15 |

## 初步模式观察

这些案例反复出现的，不是某个具体 trick，而是几类**稳定的建模对象转移动作**。

第一类是**把“状态”换成“流、算子、谱对象”**。Symplectic integrator、DMD、EDMD、Koopman MPC、DeepONet、FNO、PINO、JKO 都属于这一类：它们不再以单个时刻/单个实例为基本对象，而是把注意力提到流映射、线性传播子、解算子、分布几何或势能下降路径上。这个动作通常会带来**跨实例共享、长期可控性、分辨率不变性**。citeturn11search0turn3search0turn22search0turn22search1turn1search1turn1search5turn15search1turn13search3

第二类是**把“显式几何对象”换成“隐式场对象”**。Level set、phase field、PINN、Deep Ritz、DIMON 都是如此。界面、边界、几何变化、约束原本是 problem-specific 的烦琐细节；一旦转成 signed distance、order parameter、trial function、template-domain operator，它们就进入了统一的泛函/方程 machinery。这个模式对 ML 的启发是：很多难处理的离散结构问题，也许应先转成**连续可优化的中介场**。citeturn8search8turn8search12turn2search0turn2search1turn15search2

第三类是**把未解析复杂性显式剥离为 residual、memory、forcing 或 ensemble**。HAVOK 把混沌复杂性隔离成 forcing；Mori–Zwanzig 把消去自由度重写成 memory kernel；EKI/EnKF 把优化或过滤对象改成 ensemble flow；operator inference 把高维系统改成低维算子。这个模式很值得迁到 LLM/VLA：很多“模型失效”也许不是需要更大主干，而是需要**一个新对象专门承接被主对象排除掉的复杂性**。citeturn4search0turn7search12turn6search13turn3search3

第四类是**把全局耦合问题换成局部图/局部消息对象**。Finite volume、lattice Boltzmann、factor graph、Tanner graph、LDPC、Turbo 都说明，一个系统之所以可算，往往不是因为近似更准，而是因为对象换成了**局部可传递的 message/flux/constraint**。这对 systems、speculative decoding、multi-agent orchestration 都很关键。citeturn10search15turn11search2turn13search0turn23search8turn23search1turn23search3

第五类是**把“原始样本/轨迹”换成“慢变量、自由能面、持久结构”**。Diffusion maps、MSM、VAMP、VAMPnets、metadynamics、persistent homology、information bottleneck 都在做这件事：把最原始的数据对象压成**慢的、稳的、可比较的、决策相关的对象**。这类动作对 dataset evaluation、OOD、multimodal representation、长期记忆尤其有启发。citeturn12search1turn22search3turn22search2turn17search0turn17search5turn17search2turn12search2

最后，一个非常一致但尚未被 ML/LLM/VLA 充分吸收的观察是：**最强的进步往往不是“把已有对象拟合得更好”，而是“发明一个新的中介对象，让复杂性以更可组合的方式出现”**。在这批案例里，这些中介对象经常是：operator、belief state、transport path、memory kernel、factor graph、collective variable、template-domain map、density of states。它们看起来确实更像你说的 **taste/operator**，而不像单篇论文的小技巧。citeturn0search3turn3search3turn14search1turn13search0turn17search5turn15search2

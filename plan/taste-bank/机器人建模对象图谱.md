# **具身智能与机器人底层建模品味图谱：从状态空间到流形的范式转移**

在具身智能、机器人控制、世界模型及多模态基础模型（VLA）领域，底层建模范式的演进揭示了一个深刻的趋势：真正的突破往往并非源于网络层数的堆叠或数据集的简单扩容，而是源于“建模对象”的根本性转移（Modeling Object Transition）。研究者们通过重新定义用什么数学载体去表达物理世界的状态、机器人的动作、任务的约束或语言的意图，从而在底层逻辑上化解了旧范式难以逾越的复杂性。本报告通过深入检索与分析，构建了一份包含超过40个代表性研究案例的“建模品味图谱”，详尽梳理了这些工作如何通过改变建模对象来赋予系统全新的预测、测量与控制能力。

## **二、 从确定性点估计到分布场与流形的演进**

在机器人底层控制和物理环境感知的早期阶段，研究者通常试图寻找一个“绝对正确”的解，例如一个最优的动作向量、一个完美的抓取位姿或一个精确的物理边界框。然而，物理世界充满接触、摩擦和多模态选择。现代范式深刻地改变了这一现状，将建模对象从单一点估计转化为概率分布场或几何流形，从而赋予了模型处理不确定性和多模态表达的能力。  
在机器人策略学习领域，**Diffusion Policy** 实现了极具代表性的建模对象转移。该方法在相关论文中被详细论述（作者 Chi 等人，2023年，相关资料见于项目主页与文献库1）。在传统范式中，旧的建模对象是单一的确定性动作向量或简单高斯分布的均值。由于均值回归在遇到多峰分布（例如机器人既可以从左侧绕过障碍物，也可以从右侧绕过）时会产生“均值崩溃”现象，导致机器人直接撞向障碍物，研究者将其替换为全新的对象：**旧对象 → 新对象**：确定性动作向量 → 动作分布的分数函数梯度场（Score function / Gradient of the action distribution）。这一新对象以得分场（Score / Vector field）作为数学载体，不仅彻底移除了旧模型为了拟合多模态分布而引入混合密度网络（MDN）所带来的训练不稳定复杂性，还使研究者能够预测、测量并控制包含数十个自由度且具有高度非线性的连续动作轨迹1。该方法因此成为具身智能领域极具影响力的底层控制基石，具备极高的跨领域迁移价值，被广泛扩展至 VLA、大规模真实机器人多模态融合及长视野稳定解码中4。  
沿着类似的逻辑，**Implicit Behavioral Cloning (Implicit BC)** 重新定义了模仿学习的基础（由 Florence 等人提出，发表于 CoRL 20215）。旧的建模对象是显式的前向映射函数（输入状态，输出动作）。研究者将之改变为：**旧对象 → 新对象**：状态到动作的映射函数 → 状态-动作联合能量图景（Energy landscape）。该对象以能量场（Energy）作为数学载体，通过最小化专家动作的能量来寻优。这种对象转移消除了函数逼近器在拟合非连续边界时强加的平滑假设复杂性，使系统能够控制极其尖锐或具有离散跳跃的不连续动作空间（例如需要1毫米精度的精密插孔任务），展现了极强的决策果断性（Decisiveness）7。  
在三维空间几何与抓取生成方面，对象的转移同样剧烈。**Contact-GraspNet**（Sundermeyer 等人，2021年10）摒弃了以往预测全局对象候选边界框的做法。**旧对象 → 新对象**：全局6-DoF姿态采样 → 以点云为根节点的逐点抓取分布场。通过以标量场（Scalar Field / Point Cloud）作为数学载体，这种转移将高维度的 SE(3) 搜索空间压缩为依赖于局部物体表面的 4-DoF 空间，移除了对物体全局姿态和显式分类的依赖，使得在极度拥挤、杂乱场景下的未知物体抓取率显著提升11。相近地，**PP-GraspNet** 进一步改变了旋转矩阵的参数化方式。**旧对象 → 新对象**：具有正交约束的旋转矩阵 → 基于桌面法向量推导的指向点（Pointed Point）。这直接移除了神经网络在直接预测旋转时维持正交属性的数学约束复杂性，大幅提升了生成速度10。此外，**GIGA** 通过将离散点云转化为表达隐式几何的 TSDF 体素网格（Voxel Grid），**AO-Grasp** 将类别特征转化为局部可操作性热力图（Actionability heatmaps / Scalar Field），均是利用连续场域替代离散坐标的典型例证12。  
在更为底层的抓取扩散模型中，**Grasp Diffusion Network (GDN)** 揭示了空间拓扑特征对建模的影响14。旧的建模对象是在欧几里得空间中将平移和旋转糅合回归的坐标点。GDN 实现了：**旧对象 → 新对象**：欧几里得姿态坐标 → 解耦的李群流形扩散过程（Diffusion over Lie Group ![][image1]）。这一流形（Manifold）表达不仅移除了普通欧拉角或四元数表示旋转时固有的拓扑断裂复杂性，更使研究者能够稳定控制非单峰的多角度真实物体抓取分布14。在三维场景语义感知中，**Language Embedded Radiance Fields (LERF)**（Kerr 等人，ICCV 202315）将 **旧对象 → 新对象**：离散分类标签掩码 → 密集的、多尺度语言嵌入连续特征场。以连续向量场（Continuous Vector Field）作为载体，通过体积渲染蒸馏 CLIP 特征，移除了对显式区域提议网络的依赖，赋予了系统在 3D 空间中预测和测量开放词汇（从具体物体到如“黄色”、“电力”等抽象概念）精确分布的全新能力15。这为下游机器人的语义导航和多模态场景存储提供了无缝对接的基础。  
此外，**Transporter Networks**（Zeng 等人，CoRL 202018）堪称“操作符化”设计的典范。旧对象是物体的显式 6D 姿态坐标或三维关键点。**旧对象 → 新对象**：对象关键点与姿态坐标 → 密集空间位移特征算子场（Spatial displacements via deep feature template matching）。通过以位移算子（Operator / Vector Field）为载体，它利用了数据的三维等变性（Equivariance），彻底压缩并移除了由于遮挡或未知形态物体导致的目标检测与位姿估计的级联误差，使机器人能够在极少演示下控制形变物体（如绳索或布料）的空间重排18。

## **三、 时间序列的抽象与动作结构的聚合**

物理世界的操作并非孤立的时间切片。长期以来，强化学习和控制理论建立在马尔可夫决策过程之上，默认每一步的决策仅依赖当前状态。然而，在具有延迟反馈和摩擦力的机器人操作中，单步动作极易引发误差累积。因此，将孤立时间步聚合为块（Chunk）、序列（Sequence）或稳定的动力学系统成为了核心建模趋势。  
**Action Chunking with Transformers (ACT)**（Zhao 等人，2023年22）是这一领域的现象级工作。旧范式默认建模单步马尔可夫动作（即回答“此时此刻该输出什么扭矩”）。ACT 实施了：**旧对象 → 新对象**：单步马尔可夫动作 → 多步动作块序列与隐式风格变量。以潜在序列（Latent Sequence）为数学载体，ACT 利用条件变分自编码器（CVAE）极大压缩了高频反馈控制中微小误差随时间步指数级放大的复杂性23。它使得研究者能够将几十步操作作为一个整体运动基元来测量和控制，通过相邻块的时间集成（Temporal ensembling）实现轨迹的平滑过渡24。该方法的成功，确立了其作为微型低成本硬件（如 ALOHA 系统）及大规模 VLA 模仿学习底座的地位，广泛迁移至外科手术及双臂协作控制系统。  
**Dynamic Movement Primitives (DMP)**（Schaal 等人27）从经典控制理论出发，定义了轨迹生成的底层基石。旧的建模对象是一系列离散的空间路径点或硬编码的时间插值轨迹。**旧对象 → 新对象**：离散路径点与硬编码轨迹 → 具有收敛保证的稳定非线性动力学系统（Stable non-linear attractor system）。数学载体为基于弹簧阻尼的动力学系统（Dynamical System）。这一转移移除了由于绝对时间依赖而导致的规划刚性复杂性，使系统能够在遭遇物理扰动时自主预测自适应恢复路径，并允许动作轨迹在时间和空间尺度上自由缩放27。随后，**Language Movement Primitives (LMP)**31 进一步将高层大语言模型与 DMP 桥接。**旧对象 → 新对象**：离散符号化动作基元脚本 → VLM 直接参数化的连续 DMP 动态系统权重。它去除了符号推理与底层连续速度控制之间的割裂，提升了零样本未知环境的操作控制能力。  
在周期性运动和历史依赖控制中，**Diff-Control**（Stateful Diffusion Policy）32 提供了新的思路。**旧对象 → 新对象**：无状态孤立动作预测 → 包含内部时序一致性的控制状态（类似 ControlNet 机制的 Temporal condition）。通过状态流形（Stateful Manifold），它移除了在画圆、搅拌等周期性任务中，由于孤立观测导致模型“不知道身处周期何处”的模糊性，精准控制时序动作的连续流转32。在泛视觉序列决策上，**StARformer**33 同样将马尔可夫状态转移（**旧对象 → 新对象**：马尔可夫状态转移 → 短期状态-动作-奖励复合表征 Token 序列）作为建模载体，为基于 Transformer 的长期信用分配奠定了基础。  
这一系列在序列层面的解构，直接呼应了著名的 **Decision Transformer**33 及其衍生的基础工作 **Gato**（Reed 等人，2022年34）。Gato 将传统多任务学习中隔离的输入张量与领域特定模块，统一转移为：**旧对象 → 新对象**：领域特定输入张量 → 扁平化、统一维度的自回归标记序列（Flat sequence of tokens uniformly embedded）。通过单纯的 Token 序列数学载体，Gato 移除了为每个机器人环境单独设计感官提取器和奖励输出头的庞大工程复杂性34。这赋予了单一神经网络利用因果掩码预测并控制从雅达利游戏到真实机械臂的泛化能力，不仅证明了“万物皆可 Tokenize”的定律，更成为后继众多 VLA 与 MLLM（如 RT 系列）系统设计的精神源泉36。以此为延伸，**Octo / MMaDA-VLA** 进一步在标记空间内实现了：**旧对象 → 新对象**：连续物理控制信号 → 原生离散扩散去噪标记（Native discrete diffusion over token space）38，将视觉状态预测与动作解码完全统一在了统一的 Masked Tokens 框架内，移除了额外的前向世界模型负担。**VQ-BeT** 也在动作序列建模上采用了类似逻辑：**旧对象 → 新对象**：连续高维动作坐标 → 经由残差矢量量化的离散动作表征流形39，消除了传统 K-means 在长序列和高维空间中缺乏梯度信息与模式捕获不全的复杂性40。

## **四、 动力学分解与世界模型的解构**

构建物理世界的精确模拟器（世界模型）是具身智能的核心难题。真实环境充满了部分可观测性、摩擦力突变与非结构化物体。传统的整体式建模方式已显得力不从心，研究者开始向隐式环境表征、对象级分解与基于不确定性的预测转移。  
在腿足机器人领域，**Rapid Motor Adaptation (RMA)**（Kumar 等人，RSS 202142）促成了 Sim-to-Real 的彻底革新。旧的建模对象是直接向网络堆叠长周期的观测历史帧，或是必须通过在线系统辨识来推算的显式物理参数（如摩擦系数、载重质量）。RMA 提出了关键转移：**旧对象 → 新对象**：堆叠历史帧或显式物理参数 → 紧凑的隐式环境外参因子（Latent extrinsics / Belief state）。借由信念状态（Belief State）这一数学载体，RMA 在部署时利用一个轻量级适应网络从本体感受历史中回归该隐变量，彻底移除了在现实复杂地形（如冰面、泥地）中难以实时建立精确解析模型的复杂性46。它赋予了机器人在毫秒级时间窗内预测并适应未见地形与外部扰动的新能力，成为了四足甚至双足机器人野外控制的标准基座。受此启发，**Four-Phase Rapid Motor Adaptation (FRMA)** 进一步改进了高频控制环境下的表征方式43。**旧对象 → 新对象**：固定长度的观测历史窗口 → 经全状态监督对齐的 LSTM 细胞隐藏层动态记忆。这一时间记忆细胞（Temporal Cell）移除了固定窗口带来的内存高耗和采样策略突变带来的干扰，大幅增强了高频 POMDP 控制的鲁棒性43。  
在纯视觉驱动的世界模型中，**Slot Attention**（Locatello 等人，NeurIPS 202050）及其后续工作 **SOLD** (Slot-Attention for Object-centric Latent Dynamics)53 解决了特征纠缠问题。传统世界模型（如早期的 Dreamer）习惯于将整个画面压缩为一个全局的纠缠状态向量（Monolithic global latent state）。**旧对象 → 新对象**：整体纠缠全局隐状态 → 一组可交换的、以对象为中心的独立槽位集合。其背后的数学载体是无序集合与图结构（Set / Graph）。这种对象转移利用竞争性交叉注意力机制，移除了场景中不相关元素的非因果纠缠复杂性，使系统不再受制于场景物体的排列顺序52。通过这种解耦，研究者能在包含多个干扰物体的桌面环境中，精确预测和控制局部动力学演化，实现超强分布外（OOD）组合泛化51。顺沿这一逻辑，**COMPAS** 进一步细化了交互模式：**旧对象 → 新对象**：非因果的全局状态转移预测 → 具有因果关系的对象槽局部交互结构（Causal slot interactions）。因果图的引入有效移除了模型在长时段预测中对对象身份识别发生混淆的复杂性50。  
**TD-MPC2**（Hansen 等人，202356）对基于模型规划的架构进行了大规模缩放优化。旧的潜在动力学模型往往依赖图像解码器（Decoder）来约束潜在空间的语义对齐。TD-MPC2 实现了：**旧对象 → 新对象**：依赖解码器重建的显式状态 → 约束于单纯形的隐式无解码器表征（Decoder-free SimNorm latent representations）。通过 SimNorm 正则化构造的单纯形（Simplex）数学载体，TD-MPC2 移除了在横跨104个从简单低维到39个自由度高维任务的庞大数据集上，重建所有任务图像细节所导致的梯度爆炸与容量浪费问题58。同时，它将奖励预测从均方误差转化为离散交叉熵分布预测，极大优化了异构多任务的长期性能，成为向 VLA 和大规模多形态决策系统迁移的优秀蓝本。此外，其衍生工作 **TD-M(PC)²**61 深入诊断了规划中的价值过估计问题，将：**旧对象 → 新对象**：完全依赖探索规划策略的价值回传 → 受软约束更新修正的价值评估先验。这一调整移除了高维动作空间中因为策略不匹配导致的分布外偏差传播61。  
**Reflexive World Models (RWM)**62 同样解构了控制环路：**旧对象 → 新对象**：奖励最大化策略的直接控制输出 → 基于泰勒展开的参考轨迹跟踪残差。以模拟器残差（Simulator Residual）为载体，它成功解耦了长线任务规划与短线高频自适应之间的矛盾，使得快速控制免受模型迭代缓慢的掣肘62。  
在探讨物理不确定性控制时，Marc Deisenroth 提出的 **PILCO**（ICML 201163）依然是数据高效强化的灯塔。旧模型大多建立在确定性的状态转移方程，或忽略不确定性的参数化神经网络之上。PILCO 实施了极具先见之明的转移：**旧对象 → 新对象**：确定性系统状态转移概率 → 考虑模型认知不确定性的非参数化高斯过程（Gaussian Process）。高斯过程载体不仅提供了非线性的灵活性，最重要的是通过解析形式计算策略梯度，彻底移除了模型偏差（Model bias）带来的灾难性失败概率64。它允许研究者在只有几次物理交互数据的情境下，以严格限制系统违背安全约束的前提来优化控制输出63。这种对不确定性（Epistemic uncertainty）的精妙测算，至今对微创手术等对安全性要求极高的机器人系统具有极高的理论迁移价值。在视频生成与物理法则结合的方向上，**PhysWorld**69 把：**旧对象 → 新对象**：仅靠隐式视频表观的几何引导 → 包含交互场（InterFields）约束的物理世界重构。利用物理向量场（Vector Field）对接触关系进行监督，剔除了基于图像生成的视频经常包含的违背动力学原理的运动假象。

## **五、 语言约束、多模态对齐与任务规划的重塑**

伴随大语言模型（LLM）的崛起，机器人任务的下达和奖励设置正在从刚性代码向高度语义化流形转变。这种转移弥合了高层人类意图与底层物理执行间的巨大鸿沟。  
**VoxPoser**（Huang 等人，CoRL 202370）是零样本物理操作的突破之作。旧范式要求为不同任务辛苦收集数据以微调出专用的动作基元库（Motion primitives）。**旧对象 → 新对象**：预定义的硬编码动作基元 → 由 VLM 和 LLM 联合生成的、可组合的 3D 价值图谱（Composable 3D Value Maps：Affordance maps 和 Constraint maps）。VoxPoser 采用三维体素网格与能量场（Voxel Grid / Energy Field）作为载体，使 LLM 输出可执行代码调用 VLM 将语言中的“拉开”、“避开”转化为观测空间中密集的正负向势场分布72。这移除了需要海量操作示范数据的复杂性，并赋予了机器人无需任何额外训练，仅通过模型预测控制（MPC）结合闭环视觉反馈来处理动态扰动和物理避障的新能力72。这一转移极大拓展了 MLLM 在真实物理 3D 空间中的 Spatial Grounding 能力。  
针对操作对象的中间态几何，**Chain-of-Affordance (CoA)**76 解构了 VLA 模型的输出。**旧对象 → 新对象**：直接将语言映射到动作端点 → 包含目标定位、抓取部位、放置几何约束的中间态启示链。借助启示流形（Affordance Manifold），CoA 消除了在模糊自然语言指令下端到端黑盒决策缺乏几何先验知识的复杂性，显著增强了机器人系统在包含大量背景干扰物时的稳定表现76。对于多模态交互界面本身，**LISA (Large Language Instructed Segmentation Assistant)**（Lai 等人，CVPR 202477）实现了革命性的范式重组。旧的感知任务依赖明确指定的对象类别坐标、预定义的边界框或呆板的指示短语。**旧对象 → 新对象**：显式类别坐标与边界框回归 → “嵌入即掩码”的专用分割标记隐藏层表示（Embedding-as-mask paradigm via \<SEG\> token）。以隐含变量及标记（Latent Variable / Token）为载体，LISA 将语言理解与分割掩码解码整合，移除了繁琐的独立目标检测工程模块组合77。它不仅能直接处理“切出富含维生素C的食物”这类隐含常识推理的复杂指令，还允许通过多轮对话测量并输出像素级的精准语义图，对后续构建能够实现像素级多模态融合的 VLA 系统影响深远。  
指令形式的扩充也发生在 **VIMA (VisuoMotor Attention)** 架构中（Jiang 等人，ICML 202383）。为了开发通用机器人策略，VIMA 挑战了传统范式。**旧对象 → 新对象**：孤立离散的任务ID或纯单模态语言指令 → 跨模态交错融合的提示符序列（Multimodal prompts interleaving text and object-centric visual tokens）。借助交叉注意力层（Sequence / Cross-Attention），VIMA 将“请把该物品摆放成如【图像】所示”这类包含上下文视频/图片示例的意图完全统一。这直接移除了过去面对视觉目标导向、语言指令跟随、一次性视频模仿学习时必须训练截然不同网络结构的系统复杂性83。VIMA 借此展示了极为强大的数据缩放法则，能够在全新的材质和概念上进行零样本泛化。顺沿开放式意图，**OE-VLA (Open-Ended VLA)**87 进一步将 **旧对象 → 新对象**：固定纯语言文本指令 → 包含图像参考与多帧视频演示的开放式目标流形（Target Manifold）。其释放了模型理解物理关联（如根据白板简图绘制）的能力，打破了传统语言监督信号表达带宽不足的困局87。  
除了直接控制策略，奖励函数的设定同样经历了彻底的颠覆。**Eureka**（Ma 等人，ICLR 202488）展示了“人类级强化学习奖励设计”的未来。旧的建模对象是依赖人类专家凭经验长期试错、手动加权的标量奖励函数。**旧对象 → 新对象**：人工调节标量奖励函数 → 大语言模型驱动的、在上下文中进化的可执行奖励程序代码（Executable reward programs / Code manifold）。Eureka 采用符号化代码流形作为数学载体，利用 GPT-4 在上下文环境中自动生成、评估和结合统计学反射（Reflection）进化代码，完全去除了强化学习中最耗时的人工奖励形状塑造（Reward shaping）复杂性89。它使得研究者能够快速在各种高维异构控制（如灵巧手疾速转笔）上优化出超越人类设计的策略闭环89。这一思想极大激活了代码即策略（Code-as-policy）生态，并能够通过人类反馈的加入，平滑过渡到安全对齐（RLHF）任务中。与 Eureka 原理一脉相承的还有 **ARCHIE** 和 **PRM4RL**，它们利用 LLM 处理文本常识进而塑造出物理探索过程中的探索树或进度评估函数，甚至如 **VSPG-LLM** 介入价值重塑（Value Shaping Manifold）以替代僵化的传统层级强化学习期权更新88。  
然而，针对 Eureka 仅依据标量数值统计进行优化的盲点，**RDA (Reward Design Agent)**90 更进一步。**旧对象 → 新对象**：粗粒度的整体标量奖励统计数字反馈 → 由视觉语言模型（VLM）从行为轨迹中提取的细粒度子任务诊断残差（Semantic Residual / Text）。RDA 移除了因不同错位行为（如“将包裹扔到目标点”与“平稳推到目标点”）在奖励数值上重合导致的反常现象复杂性，实现了更符合人类真实意图的底层语义对齐控制90。在长期任务泛化中，**Voyager** 提出通过构建技能库网络（Skill Library Graph），将固定的环境状态机转移为基于互联网级知识提取的开放终身具身学习架构，彻底拓宽了强化智能体的知识边界88。最后，**Avoid Everything (MπFormer)** 避开了基于高层符号的繁琐路径规划，在安全控制上作出了基础创新。**旧对象 → 新对象**：关节配置空间碰撞搜索算法 → 端到端点云至关节位移修正算子（Point Cloud / Operator）94。它移除了需要构建密集几何地图与复杂逆运动学迭代求解的实时算力负担，让神经网络能够测量极其临近障碍物的碰撞裕度，是空间操作算子化的又一有力佐证。

## **六、 结论与初步观察**

通过对上述广泛涵盖具身基础模型、三维场景表达、机器底盘控制以及强化学习基底的数十项工作的深入拆解，我们可以提取出具身智能领域在进行底层系统创新时，几种反复出现且极具通用价值的“建模对象转移模式”：

1. **从“确定性点估计”向“分布场与能量流形”的转移 (Point to Field / Energy Landscape)**：经典模型致力于在一个坐标系或参数空间中寻找一个“绝对正确”的刚性解。然而，现代前沿模型（如 Diffusion Policy、Contact-GraspNet、LERF、Implicit BC、VoxPoser）纷纷将其替换为对全状态空间的得分场、能量地形图或放射分布建模。这种转变优雅地包容了物理互动中无处不在的多峰选择与认知不确定性。  
2. **从“显式强加几何”向“隐式结构化载体”的转移 (Explicit to Implicit Latent Structure)**：人工强加的坐标、物理方程或预定义对象边界极易产生由于系统偏差引起的级联误差或拓扑断裂（如 Gimbal Lock）。研究者（如 RMA 中的隐式外参、TD-MPC2 中的无解码器单纯形、Slot Attention 中的独立槽位集合、GDN 的解耦李群流形）选择将真实世界特征通过自监督手段，在网络深处压缩、投影至专门设计的子空间中。  
3. **从“孤立的单步状态演化”向“时空组块与因果序列”的转移 (Step-wise to Spatiotemporal Chunk / Causal Sequence)**：面对高频反馈控制中难以避免的误差累积和延迟，将独立马尔可夫决策（MDP）扩展为多步轨迹聚合（如 ACT 中的动作块、FRMA 中的时间对齐隐藏层单元、COMPAS 中的局部对象图以及 DMP 中的稳定动力学吸引子）已成为解决长时间维度时序连续性与自适应鲁棒性的黄金标准。  
4. **从“特定领域隔离向量”向“统一多模态交错标记与可微流形”的转移 (Isolated Domain-Specific to Interleaved Tokens / Code Manifolds)**：为了追求跨形态与跨任务的扩展定律（Scaling Laws），传统的隔离网络（专用的奖励计算头、专用的控制输出维、独立的状态处理网络）正在被统一。从 Gato、Octo、VIMA、LISA 展现的对文字、图像、掩码和动作信号的扁平化标记对齐，到 Eureka 依托代码这一特殊的图灵完备流形重构强化学习反馈闭环，万物归于通用生成范式的进程正前所未有地加速。
